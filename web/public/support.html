#Getting started

How do I sign in? +

GitPhotos signs in with your GitHub account. You can either use a GitHub device code, or paste a personal access token. Tap “How to get a token” on the sign-in screen for in-app instructions.

How do I create a GitHub token? +
  1. On GitHub, go to Settings → Developer settings → Personal access tokens.
  2. Create a classic token and enable the repo scope.
  3. Copy the token and paste it into GitPhotos’ sign-in screen.

The token is stored only in your device’s iOS Keychain — it never reaches us.

Do I need a paid GitHub plan? +

No. A free GitHub account includes unlimited private repositories, which is all GitPhotos needs.

#Backing up

How do I back up photos? +

Select photos and tap Add to upload them, or open Collections → Settings and turn on Auto-sync so new photos are backed up automatically.

Why is my first backup slow? +

GitPhotos uses GitHub’s standard API, which allows roughly 1,500 photos per hour. Large first-time backups simply continue across sessions until they finish.

What is auto-sharding / “storage accounts”? +

Each backup repository is capped at about 4.5 GB. When one fills up, GitPhotos automatically creates the next private repo. You can also add more GitHub accounts under Settings → Storage accounts to spread a large library across them. The primary account holds the index and can’t be removed.

The app says photo access is off. +

GitPhotos needs permission to read your library so it can back photos up. Enable it in iOS Settings → Privacy & Security → Photos → GitPhotos.

#Privacy & your data

Can GitPhotos see my photos? +

No. There is no GitPhotos server. Photos go from your device straight to your own private GitHub repositories, and your access token stays in the iOS Keychain on your phone.

How do I delete my backups or revoke access? +

Your backups are just repositories on your GitHub account:

  1. Delete photos: delete the backup repo(s) on GitHub.
  2. Cut off the app: revoke the personal access token in your GitHub settings, or sign out in the app.
